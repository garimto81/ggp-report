주요 기능

AI 기반 업무 추론

단순 키워드 매칭이 아닌, Gemini AI가 전체 대화의 맥락을 이해하고 카메라 파트 업무를 추론합니다.

쓰레드 분석

메인 메시지뿐만 아니라 쓰레드의 모든 답글까지 포함하여 완전한 대화 컨텍스트를 분석합니다.

자동 DM 전송

분석된 보고서를 여러 담당자에게 동시에 Slack DM으로 자동 전송합니다.

히스토리 관리

모든 보고서는 Supabase에 저장되어 추후 조회 및 분석이 가능합니다.

작동 방식

1

메시지 수집

Slack API를 통해 채널의 메시지와 쓰레드를 수집합니다.

2

AI 분석

Gemini AI가 문맥을 파악하여 카메라 파트 업무를 추론합니다.

3

보고서 생성

6하 원칙에 따라 업무를 정리하고 우선순위를 지정합니다.

4

DM 전송

완성된 보고서를 담당자들에게 자동으로 전송합니다.

보고서 종류

일일 보고서

실행 시간: 평일(화-금) 오전 10시

분석 범위: 최근 24시간

주요 내용:

  • 당일 진행된 촬영 업무
  • 긴급 처리 사항
  • 장비 관련 이슈

주간 보고서

실행 시간: 매주 월요일 오전 10시 10분

분석 범위: 최근 7일

주요 내용:

  • 주간 업무 요약
  • 프로젝트 진행 상황
  • 다음 주 예정 사항

월간 보고서

실행 시간: 매월 첫째 월요일 오전 10시

분석 범위: 지난달 전체

주요 내용:

  • 월간 성과 분석
  • 주요 프로젝트 완료 현황
  • 장비 운영 통계
